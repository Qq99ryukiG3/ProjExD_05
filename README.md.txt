# シューティングゲーム
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんを操作してゲームクリアを目指す。

## ゲームの実装
###共通基本機能
* Bird:こうかとんクラス
* Bobm:爆弾クラス
* Beam:ビームクラス
* Explosion:爆発クラス
* Enemy:敵クラス
* Score:スコアクラス

### 担当追加機能
* ボス戦（C0A22056）
* こうかとん強化（C0A22036）
* 敵の攻撃手段を増やすver1(C0B22122)
* 敵の攻撃手段(デバフボール)を増やす（C0B22038）
* 残機表示（C0A22160）
