# シューティングゲーム
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんを操作してゲームクリアを目指す。

## ゲームの実装
###共通基本機能
* Bird:こうかとんクラス
* Bobm:爆弾クラス
* Beam:ビームクラス
* Explosion:爆発クラス
* Enemy:敵クラス
* Score:スコアクラス
